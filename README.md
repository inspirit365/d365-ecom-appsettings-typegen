# D365 eCommerce App Settings Types
Generates TypeScript types from **Dynamics 365 eCommerce** `app.settings.json` files.

## Usage
```bash
d365-ecom-appsettings-typegen [--watch|-w] [--output|-o <output-file>]
```

## Options
- `--help`, `-h` — shows help message
- `--watch`, `-w` — optional, regenerate on file changes
- `--output`, `-o` — specify output path (default: `src/settings/app.settings.autogenerated.ts`)
